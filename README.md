# Online-OmniChord

**▶ Play Online Omnichord:** https://github.com/tunaakyurek/Online-OmniChord

This project is designed to be opened directly from GitHub Pages on iPad Safari.

## iPad user guide (player)

1. Open the Play link above in Safari (landscape).
2. Tap **Enable Audio** once.
3. Hold a chord with one finger and strum with another.
4. Tap **Rhythm Off** to enable the rhythm generator.
5. Tap **Stop All** if anything sticks.

### Add to Home Screen (optional)

1. Tap Share → **Add to Home Screen**.
2. Launch from the home screen for a full-screen experience.

### Troubleshooting

- No sound: check the mute switch and volume, then tap **Enable Audio** again.
- Strum is silent: make sure a chord is held.
- Visual misalignment: toggle **Debug Overlay** and confirm the PNG alignment.
